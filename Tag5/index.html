Um ein GitHub-Repository über SSH auf einem Mac zu verbinden, müssen Sie zunächst einen SSH-Schlüssel erstellen und ihn Ihrem GitHub-Konto hinzufügen. Hier ist eine Schritt-für-Schritt-Anleitung:

SSH-Schlüssel generieren

  1. Öffnen Sie das Terminal auf Ihrem Mac.
  2. Generieren Sie einen neuen SSH-Schlüssel mit folgendem Befehl: 
    bash
    ssh-keygen -t ed25519 -C "ihre_email@example.com"
    Ersetzen Sie "ihre_email@example.com" durch die E-Mail-Adresse, die Sie für Ihr GitHub-Konto verwenden
    1
    2
    .
  3. Wenn Sie zur Eingabe eines Dateinamens aufgefordert werden, drücken Sie einfach Enter, um den Standardspeicherort zu akzeptieren.
  4. Sie können optional eine Passphrase für zusätzliche Sicherheit eingeben oder einfach Enter drücken, um keine Passphrase zu verwenden
    1
    .

SSH-Schlüssel zum SSH-Agent hinzufügen

  1. Starten Sie den SSH-Agent im Hintergrund: 
    bash
    eval "$(ssh-agent -s)"
  2. Fügen Sie Ihren privaten SSH-Schlüssel zum SSH-Agent hinzu: 
    bash
    ssh-add --apple-use-keychain ~/.ssh/id_ed25519
    Wenn Sie keine Passphrase verwendet haben, lassen Sie die Option --apple-use-keychain weg
    5
    .

SSH-Schlüssel zu GitHub hinzufügen

  1. Kopieren Sie den öffentlichen SSH-Schlüssel in die Zwischenablage: 
    bash
    pbcopy < ~/.ssh/id_ed25519.pub
  2. Melden Sie sich bei GitHub an und navigieren Sie zu "Settings" > "SSH and GPG keys"
    2
    4
    .
  3. Klicken Sie auf "New SSH key" oder "Add SSH key"
    2
    4
    .
  4. Geben Sie einen beschreibenden Titel für den Schlüssel ein und fügen Sie den kopierten öffentlichen Schlüssel in das Feld "Key" ein
    2
    4
    .
  5. Klicken Sie auf "Add SSH key", um den Vorgang abzuschließen
    2
    4
    .

SSH-Verbindung testen

Um sicherzustellen, dass alles korrekt eingerichtet wurde, können Sie die SSH-Verbindung zu GitHub testen: 
bash
ssh -T git@github.com
Wenn alles funktioniert hat, sollten Sie eine Begrüßungsnachricht von GitHub sehen
1
4
.

Repository klonen

Nachdem Sie Ihren SSH-Schlüssel eingerichtet haben, können Sie nun Repositories über SSH klonen. Verwenden Sie dazu den SSH-URL des Repositories: 
bash
git clone git@github.com:benutzername/repository.git
Ersetzen Sie "benutzername" und "repository" durch die entsprechenden Werte für Ihr Repository
4
. Mit diesen Schritten sollten Sie nun in der Lage sein, GitHub-Repositories über SSH auf Ihrem Mac zu verwenden. Dies ermöglicht eine sichere Verbindung ohne die Notwendigkeit, bei jedem Push oder Pull Ihren Benutzernamen und ein persönliches Zugriffstoken einzugeben.
